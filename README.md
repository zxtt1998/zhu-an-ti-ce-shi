# （自用）注安题测试

注册安全工程师《安全生产技术基础》手机刷题网页。

## 功能

- 章节练习、错题强化、模拟考试
- 180 道内置练习题
- 新增“教材参考”入口，按 2026 官方教材重整题库边界
- 参考 2020-2025 年真题出题方式整理场景题、规范题和辨析题
- 两种判题方式：
  - 答完立刻看答案
  - 本小节或本章结束后统一对答案
- 每题可记录本地笔记
- 错题支持分享到 iPhone 备忘录、复制或下载 Markdown 文件
- 适配 iPhone 手机刷题
- 数据保存在浏览器本地

## 部署

这是纯静态网页，推送到 GitHub 后可用 GitHub Pages 部署。Pages 来源选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
